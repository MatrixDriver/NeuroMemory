Forgetting
会遗忘的记忆系统
只增不减会导致信息过载。neuromem 拥有四重遗忘机制，让记忆保持高信噪比。
⏳

时间衰减

基于 Ebbinghaus 遗忘曲线。长期无强化的特质置信度自然下降。behavior 衰减快，core 衰减慢。

🔄

间隔效应

被反复验证的记忆更抗衰减。强化 10 次后衰减速度减半，模拟人类间隔重复学习。

⚡

矛盾反思

新证据与已有特质矛盾时，触发专项审查。LLM 综合判断：修正 / 情境化分裂 / 废弃。

🪟

趋势过期

trend 阶段用时间窗口管理。观察期内未被强化的趋势自动清除，不留垃圾。

Advantages
核心优势
01

🛡️ 防表演污染

用户可能在 AI 面前"表演人设"。neuromem 要求特质经多次跨对话验证才能确立。研究显示 LLM 单次推断人格准确度仅 r=0.27 —— 所以我们不这么做。

02

🔗 全链路可追溯

每条特质保留完整证据链。哪些事实支持、哪些矛盾，一目了然。升级后底层记忆不删除，随时可回溯到原始证据重新评估。

03

🧠 认知科学驱动

不是拍脑袋设计。每个核心决策都有认知心理学理论支撑：Tulving 记忆分类、Conway 自我记忆系统、Ebbinghaus 遗忘曲线、Mischel 情境人格理论。

04

🌍 全球独创架构

覆盖 Stanford Generative Agents、MemGPT、Mem0、Zep 等全球主流系统的调研中，trait 三层自动升级链和四重遗忘机制均为独一无二。
